install:

-------------------------------------------
remove the original installation
sudo apt-get purge wiringpi

-------------------------------------------
check update:

cd ~/wiringPi
git pull origin

-------------------------------------------
build/install:

cd ~/wiringPi
./build

-------------------------------------------
Test wiringPi’s installation

gpio -v
gpio readall

That should give you some confidence that it’s working OK
